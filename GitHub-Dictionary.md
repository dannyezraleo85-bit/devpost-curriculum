# GitHub Dictionary

Welcome to the GitHub Dictionary! Here you will find a comprehensive list of GitHub terminology with definitions, emojis, and translations in multiple languages.

### A
- **Action** 🤖  
  A workflow that is run in response to specific events.  
  - **Spanish**: Acción  
  - **French**: Action  
  - **German**: Aktion  
  - **Chinese**: 动作  
  - **Japanese**: アクション  

### B
- **Branch** 🌿  
  A parallel version of a repository.  
  - **Spanish**: Rama  
  - **French**: Branche  
  - **German**: Zweig  
  - **Chinese**: 分支  
  - **Japanese**: ブランチ  

### C
- **Commit** 💾  
  Saving changes to a repository.  
  - **Spanish**: Confirmar  
  - **French**: Valider  
  - **German**: Commit  
  - **Chinese**: 提交  
  - **Japanese**: コミット  

### D
- **Fork** 🍴  
  A copy of a repository that you can modify.  
  - **Spanish**: Bifurcación  
  - **French**: Fork  
  - **German**: Fork  
  - **Chinese**: 派生  
  - **Japanese**: フォーク  

### E
- **Issue** 🐛  
  A way to track tasks, enhancements, and bugs.  
  - **Spanish**: Problema  
  - **French**: Problème  
  - **German**: Problem  
  - **Chinese**: 问题  
  - **Japanese**: 課題  

### M
- **Merge** 🔄  
  Combining changes from different branches.  
  - **Spanish**: Fusionar  
  - **French**: Fusionner  
  - **German**: Zusammenführen  
  - **Chinese**: 合并  
  - **Japanese**: マージ  
  
### P
- **Pull Request** 📥  
  A request to merge changes from one branch to another.  
  - **Spanish**: Solicitud de extracción  
  - **French**: Demande de tirage  
  - **German**: Pull-Request  
  - **Chinese**: 拉取请求  
  - **Japanese**: プルリクエスト  

### R
- **Repository** 📂  
  A storage space for your project files.  
  - **Spanish**: Repositorio  
  - **French**: Dépôt  
  - **German**: Repository  
  - **Chinese**: 仓库  
  - **Japanese**: リポジトリ  

### S
- **Star** ⭐  
  A way to bookmark repositories.  
  - **Spanish**: Estrella  
  - **French**: Étoile  
  - **German**: Stern  
  - **Chinese**: 星标  
  - **Japanese**: スター  

### W
- **Workflow** 🚦  
  A configurable automated process.  
  - **Spanish**: Flujo de trabajo  
  - **French**: Flux de travail  
  - **German**: Arbeitsablauf  
  - **Chinese**: 工作流  
  - **Japanese**: ワークフロー  

Feel free to contribute by adding more terms to the dictionary!
